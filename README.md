# 🌿 Medicinal Plant Identification System

## 💡 Overview
An AI-powered web application that identifies medicinal plants from images using machine learning. The system provides detailed information including scientific names, medicinal uses, and health benefits, promoting awareness of natural remedies.

## 🚀 Features
- 📷 Upload plant images for instant identification  
- 🤖 Machine learning-based classification  
- 🌱 Informational database (scientific name, uses, benefits)  
- 🎨 Clean and responsive user interface  
- ⚡ Fast prediction results with visualization  

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend/ML: Python  
- Machine Learning: Image Classification  

## 🎯 Purpose
To combine AI and web technology for educational and healthcare awareness applications.
